For maunal installation of python.

Perform following steps.

* Following step is only for the python 3.6.5(64 bit) installation. 
  download directly from https://www.python.org/ftp/python/3.6.5/python-3.6.5-amd64.exe

1) Python installation

Perform manual installation under the location C:/Python/Python3.6.5

-> Add to the enviroment variable under PATH, the above directory so that python
   code can be executed from the command prompt.


2) Pip installation to manage the packages and libraries.

  -> Go to the link https://bootstrap.pypa.io/get-pip.py.
  -> Copy and paste the code in notepad and save it as "get-pip.py" and save into
     python installation directory.
  ->  Now open command prompt and type "python get-pip.py" 


-> Add pip to the enviroment variable by searching for "pip" in installation 
   directory. Copy the path and add to the PATH in enviroment variable.

-> Perform following command for library installation
   pip install numpy
   pip install scipy
   pip install matplotlib
   pip install scikit-learn
   pip install opencv-python
   pip install cmake //cmake is a standard c++ support for python
   pip install dlib //requires cmake installation before



-> Concern https://stackoverflow.com/questions/41912372/dlib-installation-on-windows-10 for installation error on windows 10 machines
